## IV. 회의 내용

### 1. 팀원별 주제 아이디어 공유

- ① 이온 — GitHub 기여도 분석 기반 채용 스크리닝 서비스  
  - GitHub URL 입력 시 커밋 일관성·품질, 실질적 기여도, 순환 복잡도 등 8개 알고리즘으로 개발자 역량을 자동 분석·점수화  
  - GitHub GraphQL API + Redis 캐싱, Gemini API 활용 하이브리드 설계 (알고리즘이 점수 산출, AI는 자연어 표현만 담당)  

- ② 이은성 — Grad-Deploy (컨테이너 기반 인프라 자동 구축 솔루션)  
  - Docker Compose + Dockerfile을 GUI 기반 마법사로 자동 생성하고 GitHub에 원클릭 Push하는 서비스  
  - K8s YAML 자동 생성, 리소스 모니터링 대시보드, 데모데이 모드 등 졸업작품 특화 기능 포함  

- ③ 이혁제 — 자세 분석 운동 추천 서비스  
  - 유사 서비스가 이미 존재하는 것으로 확인되어 차별화 어렵다고 판단, 스스로 보류  

- ④ 고민서 — 이은성 제안 주제에 찬성
